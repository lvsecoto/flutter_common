和业务无关的通用组件

## 特点

1. `common`是和业务，品牌无关的通用组件，比如对`riverpod`的功能，图片选择器，也包括对Material Design的扩展
2. 可利用全局主题来定制组件

## 开始

把这个包的github地址引入到pubspec.yaml

```yaml
common:
  git:
    url: git@codeup.aliyun.com:6240585956f85235f7dd77d1/app/flutter_common.git
    ref: [指定版本]
```
